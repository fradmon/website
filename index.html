Section 3: Filing a New Claim
3.1 Navigate to the Claims_DB folder located in the left-hand sidebar of the application.
3.2 Verify the insured party's name and select the appropriate Coverage Code from the dropdown menu (e.g., STD-LIABILITY-A).
3.3 Review all line items in the cost table. Ensure each entry matches the adjuster's report before proceeding.
3.4 Click Process to submit the claim for approval. A confirmation dialog will appear.
Note: Claims exceeding $5,000 require supervisor approval before processing. See Section 5 for escalation procedures.
